# CATENA — Personal AI Operating System

> Chain every tool. Automate every task. Own every outcome.

## Architecture
INPUT → UNDERSTAND → DECIDE → ACT → STORE → IMPROVE

## Stack
| Layer | Tool |
|---|---|
| Automation Engine | n8n (self-hosted) |
| Server | Hetzner VPS (Ubuntu 22.04) |
| Ticketing | Freshdesk |
| Monitoring | Prometheus |
| Communication | Microsoft Teams + Outlook |
| Tasks | Google Tasks |
| Command Channel | Telegram |

## Phases
- **Phase 1** ✅ Foundation — Server + n8n + input channels + daily brief
- **Phase 2** 🔜 Freshdesk automation + SLA prediction + Teams co-pilot
- **Phase 3** 🔮 AI agents + knowledge brain + opportunity radar
